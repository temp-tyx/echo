### 2.2 `_apply_echo_pruning`

在 Eagle propose 返回前调用（`VLLM_ECHO_ENABLED` 时）：

1. 从 `_echo_logits_list` 取每步 logits，算条件 log-prob
2. `cumsum` 得 `cum_log_probs`
3. **NaN 防护**：非有限值替换为 `-inf`，`k_max` 限制为有限值个数
4. global `torch.topk` 选 top-k 单元格
5. 未选中位置设 `-1`

```python
cond_log_probs = torch.where(
    torch.isfinite(cond_log_probs),
    cond_log_probs,
    torch.tensor(float("-inf"), device=..., dtype=...),
)
cum_log_probs = torch.cumsum(cond_log_probs, dim=1)

flat_log_probs = cum_log_probs.flatten()
num_finite = int(torch.isfinite(flat_log_probs).sum().item())
k_max = min(envs.VLLM_ECHO_K_MAX, num_finite)

flat_for_topk = flat_log_probs.clone()
flat_for_topk[~torch.isfinite(flat_for_topk)] = float("-inf")
_, top_flat_indices = torch.topk(flat_for_topk, k=k_max)
```

在 `_update_states` / `_prepare_inputs` 之前裁剪 scheduler 输出。

### 3.3 `_apply_echo_scheduler_trim`

对每个 `req_id`：

1. 从 `echo_cu_draft_tokens` / `echo_draft_tokens` 取 keep 信息
2. `kept_spec_tokens = proposed_drafts[:draft_token_num]`
3. 更新 `scheduled_spec_decode_tokens`、`num_scheduled_tokens`
4. 重算 `total_num_scheduled_tokens`

若 `kept_spec_tokens` 为空，从 `scheduled_spec_decode_tokens` 删除该 req。

### 3.4 `_prepare_input_ids`（override）

调用 `super()._prepare_input_ids(...)` 后，ECHO 路径额外：

1. 将 trim 后的 `spec_tokens` 写入 `input_ids_cpu[spec_start:flat_end+1]`
2. 从 `prev_sampled_token_ids` 写入 sample 位置
3. `copy_to_gpu`

**动机**：async spec-decode 的 base scatter 无法正确处理 ECHO trim 后的变长 draft，导致 `input_ids` 为 0 → hidden NaN。

**TypeError 修复**：赋值须用 `torch.tensor(spec_tokens, dtype=input_ids_cpu.dtype)`，不能用 `np.asarray`。

### 3.5 `_get_draft_token_ids_cpu`（override）

ECHO 启用时过滤 `-1`，返回 `(filtered_rows, req_ids)`。

### 3.6 `propose_draft_token_ids` 闭包内

propose 完成后：

```python
mask = (self._draft_token_ids != -1)
kept_counts = mask.int().sum(dim=1).cpu().tolist()
filtered_drafts, draft_req_ids = self._get_draft_token_ids_cpu()
req_ids_for_echo = draft_req_ids if draft_req_ids else self.input_batch.req_ids
self.echo_cu_draft_tokens = dict(zip(req_ids_for_echo, kept_counts))
self.echo_draft_tokens = dict(zip(req_ids_for_echo, filtered_drafts))
```
