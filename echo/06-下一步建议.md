## 优先级3: 定位NaN首次出现的位置

在`_model_forward`内添加更细粒度的检查:

```python
# 在model_runner_v1.py的_model_forward中
hidden_states = self.model(...)

# 按层检查
for i, layer_output in enumerate(hidden_states):
    if torch.isnan(layer_output).any():
        logger.info(f"ECHO [layer_nan] layer={i} req_idx=...")
```

确定是:
- 第一层就出现NaN (输入问题)
- 中间某层出现NaN (attention计算问题)
- 只有最后一层NaN (可能不是真NaN，是数值问题)
